## MS_TO_KMH

| | |
|:---|:---|
| **Type	Funktion** | REAL |
| **Input	MS** | REAL (Geschwindigkeit in km/h) |
| **Output** | REAL (Geschwindigkeit in m/s) |
| | MS_TO_KMH rechnet einem Geschwindigkeitswert von Meter / Sekunde in Kilometer / Stunde um. |
| **MS_TO_KMH** | = MS * 3.6 |
