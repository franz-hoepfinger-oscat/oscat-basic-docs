## KMH_TO_MS

| | |
|:---|:---|
| **Type	Funktion** | REAL |
| **Input	KMH** | REAL (Geschwindigkeit in m/s) |
| **Output** | TIME (Geschwindigkeit in km/h) |
| **KMH_TO_MS rechnet einem Geschwindigkeitswert von Kilometer / Stunde in Meter / Sekunde um.	KMH_TO_MS** | = KMH / 3.6 |
