## PT_TO_F

| | |
|:---|:---|
| **Type	Funktion** | REAL |
| **Input	PT** | TIME (Periodendauer in Sekunden) |
| **Output** | REAL (Frequenz in Hz) |
| | PT_TO_F rechnet eine Periodendauer in Sekunden in die entsprechende Frequenz in Hz um. |
