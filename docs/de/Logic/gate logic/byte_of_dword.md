## Type	Funktion : BYTE

| | |
|:---|:---|
| **Input	IN** | DWORD (Eingangs-DWORD) |
| **Output** | BYTE (Ausgangsbyte) |
| | BYTE_OF_DWORD extrahiert ein Byte (B0 .. B3) aus einem DWORD. Die einzelnen Bytes werden mit 0 - 3 am Eingang IN ausgewählt. |
