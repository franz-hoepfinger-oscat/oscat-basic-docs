## DWORD_OF_WORD

| | |
|:---|:---|
| **Type	Funktion** | DWORD |
| **Input	W1** | WORD (Eingangs WORD 1) |
| **W0** | WORD (Eingangs WORD 0) |
| **Output** | DWORD (Ergebnis-DWORD) |
| | DWORD_OF_WORD setzt aus 2 separaten WORDS W0 und W1 ein DWORD zusammen. |
| **Ein DWORD setzt sich zusammen wie folgt** | W1-W0. |
