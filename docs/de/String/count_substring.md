## COUNT_SUBSTRING

| | |
|:---|:---|
| **Type	Funktion** | INT |
| **Input	SEARCH** | STRING (zu untersuchende Zeichenkette) |
| **STR** | STRING (Suchstring) |
| **Output** | INT (Anzahl der Fundstellen) |
| | COUNT_SUBSTRING ermittelt wie oft ein Substring STR im String SEARCH vorkommt. |
