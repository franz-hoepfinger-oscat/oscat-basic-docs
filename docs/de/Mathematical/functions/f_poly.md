## Type	Funktion : REAL

| | |
|:---|:---|
| **Input	X** | REAL (Eingang) |
| **C** | ARRAY[0..7] of REAL (Polynomkoeffizienten) |
| **Output** | REAL (Ergebnis der Polynomgleichung) |
| | F_POLY errechnet ein Polynom 7ten Grades. |
| | F_POLY = C[0] + C[1] * X^1 + C[2] * X^2 + ...C[7] * X^7 |
