## CIRCLE_SEG

| | |
|:---|:---|
| **Type	Funktion** | REAL |
| **Input	RX** | REAL	(Kreisradius) |
| **HX** | REAL	(Höhe der Sektantlinie) |
| **Output	Real** | (Fläche des Segments) |
| | CIRCLE_SEG berechnet die Fläche eines Kreissegments das durch eine Sektantlinie und den Kreis eingeschlossen ist. |
