## GCD

| | |
|:---|:---|
| **Type** | Funktion |
| **Input	A** | DINT (Eingangswert A) |
| **B** | DINT (Eingangswert B) |
| **Output** | INT (Größter gemeinsamer Teiler) |
| | Die Funktion GCD berechnet den größten gemeinsamen Teiler (GGT) von A und B. |
