## Type	Funktion : DINT

| | |
|:---|:---|
| **Input	X** | REAL (Eingangswert) |
| **Output** | DINT (Ausgangswert) |
| | Die Funktion FLOOR2 gibt den größten ganzzahligen Wert kleiner gleich X zurück. |

![floor2](floor2.gif)

**Beispiel:**

```iecst
FLOOR2(3.14) = 3 FLOOR2(-3.14) = -4 FLOOR2(2) = 2
```
