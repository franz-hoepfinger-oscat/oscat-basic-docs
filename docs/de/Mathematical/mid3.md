## MID3

| | |
|:---|:---|
| **Type	Funktion** | REAL |
| **Input	IN1** | REAL (Eingang 1) |
| **IN2** | REAL (Eingang 2) |
| **IN3** | REAL (Eingang 3) |
| **Output** | REAL (Mittlerer Wert der 3 Eingänge) |
| | Die Funktion MID3 liefert den mittleren Wert von 3 Eingängen, nicht aber den Mathematischen Mittelwert. |

![mid3](mid3.gif)

**Beispiel:**

```iecst
MID3(1,5,2) = 2.
```
