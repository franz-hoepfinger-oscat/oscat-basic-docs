## Type	Funktion : DINT

| | |
|:---|:---|
| **Input	X** | REAL (Eingangswert) |
| **Output** | DINT (Ausgangswert) |
| | Die Funktion CEIL2 gibt den kleinsten ganzzahligen Wert größer gleich X zurück. |

![ceil2](ceil2.gif)

**Beispiel:**

```iecst
CEIL2(3.14) = 4 CEIL2(-3.14) = -3 CEIL2(2) = 2
```
