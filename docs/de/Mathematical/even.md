## Type	Funktion : BOOL

| | |
|:---|:---|
| **Input	in** | DINT (Eingangswert) |
| **Output** | BOOL (TRUE, wenn in gerade) |
| | Die Funktion EVEN wird TRUE, wenn der Eingang IN gerade ist und FALSE für ungerade IN. |

![even](even.gif)

**Beispiel:**

```iecst
EVEN(2) ergibt TRUE EVEN(3) ergibt FALSE
```
