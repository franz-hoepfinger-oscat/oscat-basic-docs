## MUL_ADD

| | |
|:---|:---|
| **Type	Funktion** | REAL |
| **Input	X** | REAL (Eingangswert) |
| **K** | REAL (Multiplikator) |
| **O** | REAL (Offset) |
| **Output** | REAL (Ausgangswert) |
| | MUL_ADD multipliziert den Eingangswert X mit K und addiert O hinzu. |
| | MUL_ADD = X * K + O. MUL_ADD(0.5, 10, 2) ergibt 0.5 * 10 + 2 = 7. |
