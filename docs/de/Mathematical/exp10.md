## Type	Funktion : REAL

| | |
|:---|:---|
| **Input	X** | REAL (Eingangswert) |
| **Output** | REAL (Exponentialfaktor zur Basis 10) |
| | Die Funktion EXP10 liefert den Exponentialwert zur Basis 10. |
| | EXP10(2) = 100 |
| | EXP10(0) = 1 |
| | EXP10(3.14) = 1380.384265 |
