## Type	Funktion : INT

| | |
|:---|:---|
| **Input	X** | REAL (Eingangswert) |
| **Output** | INT (Ausgangswert) |
| | Die Funktion FLOOR gibt den größten ganzzahligen Wert kleiner gleich X zurück. |

![floor](floor.gif)

**Beispiel:**

```iecst
FLOOR(3.14) = 3 FLOOR(-3.14) = -4 FLOOR(2) = 2
```
