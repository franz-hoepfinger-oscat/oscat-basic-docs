## SECOND_OF_DT

| | |
|:---|:---|
| **Type	Funktion** | INT |
| **Input	XDT** | DATETIME (Eingangswert) |
| **Output** | INT (aktuelle Sekunde) |
| | SECOND_OF_DT extrahiert die momentane Sekunde aus einem DT Wert. |
| **SECOND_OF_DT(DT#2008-6-6-10** | 22:20) = 20 |
