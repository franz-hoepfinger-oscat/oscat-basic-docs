## Type	Funktion : INT

| | |
|:---|:---|
| **Input	XDT** | DATETIME (Eingangswert) |
| **Output** | INT (aktuelle Minute) |
| | MINUTE_OF_DT extrahiert die momentane Minute aus einem DT Wert. |
| **MINUTE_OF_DT(DT#2008-6-6-10** | 22:20) = 22 |
