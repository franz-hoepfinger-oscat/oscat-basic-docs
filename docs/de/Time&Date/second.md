## Type	Funktion : REAL

| | |
|:---|:---|
| **Input	ITOD** | TOD (Tageszeit) |
| **Output** | REAL (Sekunden und Millisekunden der Tageszeit) |
| | Die Funktion SECOND extrahiert den Sekundenanteil aus der Tageszeit |

![second](second.gif)

**Beispiel:**

```iecst
SECOND(22:10:12.331) = 12.331
```
