## Die Struktur VECTOR_3 speichert die 3 Vektoren eines dreidimensionalen Vektorraums.

| | | |
|:---|:---|:---|
| ***.X** | REAL | Komponente in X - Richtung |
| ***.Y** | REAL | Komponente in Y - Richtung |
| ***.Z** | REAL | Komponente in Z - Richtung |
