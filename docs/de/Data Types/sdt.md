## Die Struktur SDT definiert ein strukturiertes Zeit- Datums- Format.

| | | |
|:---|:---|:---|
| ***.YEAR** | INT | Jahr |
| ***.MONTH** | INT | Monat |
| ***.DAY** | INT | Tag |
| ***.WEEKDAY** | INT | Wochentag (1 = Montag, 7 = Sonntag) |
| ***.HOUR** | INT | Stunden |
| ***.MINUTE** | INT | Minuten |
| ***.SECOND** | INT | Sekunden |
| ***.MS** | INT | Millisekunden |
