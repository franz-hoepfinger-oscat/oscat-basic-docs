## FRACTION

| | | |
|:---|:---|:---|
| ***.NUMERATOR** | INT | Zähler des Bruches (Numerator) |
| ***.DENOMINATOR** | INT | Nenner des Bruches (Denumerator) |

Der Datentyp FRACTION kann benutzt werden um einen Bruch zu Repräsentieren.
