## Type	Function: REAL

| | |
|:---|:---|
| **Input	ITOD** | TOD (time of day) |
| **Output** | REAL (seconds and milliseconds of time of day) |
| | The function  SECOND  extracts the seconds portion of the day |

![second](second.gif)

**Example:**

```iecst
SECOND(22:10:12.331) = 12.331
```
