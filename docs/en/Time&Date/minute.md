## MINUTE

| | |
|:---|:---|
| **Type	Function** | INT |
| **Input	ITOD** | TIMEOFDAY (day time) |
| **Output** | INT (current minute) |
| | The MINUTE function extracts the current minute of the day. |

![minute](minute.gif)

**Example:**

```iecst
MINUTE(22:55:13) = 55
```
