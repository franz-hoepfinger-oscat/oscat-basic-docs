## Type	Function: INT

| | |
|:---|:---|
| **Input	IDATE** | DATE (date) |
| **Output** | INT (year of the date) |
| | The function YEAR_OF_DATE calculates the corresponding year from the date of input IDATE. |

![year_of_date](year_of_date.gif)

**Example:**

```iecst
YEAR_OF_DATE(31.12.2007) = 2007
```
