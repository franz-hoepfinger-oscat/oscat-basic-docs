## Type	Function: INT

| | |
|:---|:---|
| **Input	ITOD** | TIMEOFDAY (day time) |
| **Output** | INT (current hour) |
| | The HOUR function extracts the current hour of the day. |

![hour](hour.gif)

**Example:**

```iecst
HOUR(22:55:13) = 22
```
