## PARITY

| | |
|:---|:---|
| **Type	Function** | BOOL |
| **Input	IN** | BYTE (BYTE input) |
| **Output** | BOOL (output is TRUE if parity is even) |
| | PARITY calculates even parity over the input byte IN. The output is TRUE if the number of true bits in the byte (IN) is odd. |
