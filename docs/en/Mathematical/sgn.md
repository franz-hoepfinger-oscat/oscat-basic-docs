## Type	Function: INT

| | |
|:---|:---|
| **Input	X** | REAL (input) |
| **Output** | INT (  Signum  the input X) |
| | The function SGN  calculates the  Signum  of X. |
| | SGN = +1 if X > 0 |
| | SGN = 0 if X = 0 |
| | SGN = -1 if X < 0 |
