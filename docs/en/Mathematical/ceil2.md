## Type	Function: DINT

| | |
|:---|:---|
| **Input	X** | REAL (input) |
| **Output** | DINT (output value) |
| | The function returns the smallest integer value CEIL2 greater or equal to X. |

![ceil2](ceil2.gif)

**Example:**

```iecst
CEIL2(3.14) = 4 CEIL2(-3.14) = -3 CEIL2(2) = 2
```
