## Type	Function: DINT

| | |
|:---|:---|
| **Input	X** | REAL (input) |
| **Output** | DINT (output value) |
| | The function floor2 returns the largest integer value less or equal than X back. |

![floor2](floor2.gif)

**Example:**

```iecst
FLOOR2(3.14) = 3 FLOOR2(-3.14) = -4 FLOOR2(2) = 2
```
