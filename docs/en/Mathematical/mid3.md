## MID3

| | |
|:---|:---|
| **Type	Function** | REAL |
| **Input	IN1** | REAL (input 1) |
| **IN2** | REAL (input 2) |
| **IN3** | REAL (input 3) |
| **Output** | REAL (mean value of the 3 inputs) |
| | The function  MID3 returns the average value of 3 inputs, but not the mathematical average. |

![mid3](mid3.gif)

**Example:**

```iecst
MID3(1,5,2) = 2.
```
