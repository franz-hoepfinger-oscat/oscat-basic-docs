## LAMBERT_W

| | |
|:---|:---|
| **Type	Function** | REAL |
| **Input	X** | REAL (input) |
| **Output** | REAL (output value) |
| | The LAMBERT_W function is defined for x > = -1/e. When the value is below the range the is result -1000. The range of LAMBERT_W function is >= -1. |
