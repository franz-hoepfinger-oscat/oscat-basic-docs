## Type	Function: STRING

| | |
|:---|:---|
| **Input	STR** | STRING (String input) |
| **Output** | STRING (STR without spaces) |
| | The TRIM function removes all spaces from STR. |

![trim](trim.gif)

**Example:**

```iecst
TRIM('find BX12') = 'findBX12'
```
