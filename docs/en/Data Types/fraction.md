## FRACTION

| | | |
|:---|:---|:---|
| ***.NUMERATOR** | INT | Numerator of the fraction (numerator) |
| ***.DENOMINATOR** | INT | Denominator of the fraction (Denumerator) |

The data type FRACTION can be used to represent a fraction .
