## The structure  for SDT defines a structured time-date format.

| | | |
|:---|:---|:---|
| ***.YEAR** | INT | Year |
| ***.MONTH** | INT | Month |
| ***. DAY** | IN  T | Day |
| ***. HOUR** | INT | Hours |
| ***. MINUTE** | INT | Minutes |
| ***. SECOND** | INT | Seconds |
| ***. MS** | INT | Milliseconds |

WEE *.  KDAY: INT		Week (1 = Monday, 7 = Sunday)
