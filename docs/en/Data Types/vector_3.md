## The  Structure  VECTOR_3 stores the three vectors of a three-dimensional vector space.

| | | |
|:---|:---|:---|
| ***.X** | REAL | component in x direction |
| ***.Y** | REAL | Component in Y - direction |
| ***.Z** | REAL | Component in Z - direction |
