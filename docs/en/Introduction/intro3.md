## Vectors are mapped to the defined type VEKTOR_3.

The type VEKTOR_3 consists of 3 components X, Y and Z, all components are of type REAL. The vector of type vector V consists of: V.X	X component of the type REAL. V.Y	Y component of the type REAL. V.Z	Z component of the type REAL.
