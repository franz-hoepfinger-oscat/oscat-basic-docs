## Demo programs

The OSCAT  Network  Library contains components and functions that deal with the issue of file-handling and ethernet communications, and sometimes require enhanced base knowledge . In order to allow the user easy access are possible, for many   theme demo programs are prepared. The demo programs are in network.lib in the folder "DEMO" included. If they are used, the needed programs should be copied to your project and then can be adjusted to your needs. Some modules require the disclosure of its own specific parameters so that they are fully functional. The Codesys and Beckhoff library demo programs are hidden because they would otherwise occupy resources needlessly.
