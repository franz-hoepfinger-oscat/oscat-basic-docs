## Complex numbers are shown with the defined type [COMPLEX](Data Types/complex.md).

The type [COMPLEX](Data Types/complex.md) consists of a real part and an imaginary part, both components are of type REAL. The complex number Z of type [COMPLEX](Data Types/complex.md) consists of: *.RE	Real part of type REAL. *.IM	Imaginary part of the type REAL.
